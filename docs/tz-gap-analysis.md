| Что | Почему |
|-----|--------|
| SVG-слой карты | Жёсткое ограничение ТЗ |
| Логика фильтрации карты | Не влияет на пользовательские экраны |
| Структура map-screen | Только контейнерная интеграция |
| Серверная архитектура | За рамками фронт-прототипа |
| ИБ-контуры, сетевые схемы | Не влияют на UI |
